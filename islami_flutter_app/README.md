# منصة إسلامي - تطبيق Flutter

تطبيق إسلامي شامل مبني بتقنية Flutter، يحول موقع "منصة إسلامي" إلى تطبيق موبايل للأندرويد والآيفون.

## 📱 الميزات

- 🕌 مواقيت الصلاة مع الإشعارات
- 📖 القرآن الكريم مع تتبع الختمات
- 📿 الأذكار اليومية
- 📚 مقالات إسلامية مصنفة
- 🤲 مكتبة الأدعية
- 📜 قصص الأنبياء
- 🌿 الطب النبوي
- 🎙️ البودكاست الإسلامي
- 📻 إذاعة مباشرة
- 🧮 حاسبة الزكاة والمواريث
- 🕋 اتجاه القبلة
- 📅 التقويم الهجري
- 🌙 الوضع الليلي

## 🚀 التثبيت

### المتطلبات
- Flutter SDK 3.0 أو أحدث
- Android Studio / Xcode
- حساب مطور (للنشر على المتاجر)

### خطوات التثبيت

1. **تحميل المشروع**
```bash
cd islami_flutter_app
```

2. **تثبيت المكتبات**
```bash
flutter pub get
```

3. **تشغيل التطبيق**
```bash
# للأندرويد
flutter run -d android

# للآيفون (macOS فقط)
flutter run -d ios
```

## 📦 البناء والنشر

### للأندرويد (APK)
```bash
flutter build apk --release
```

### للأندرويد (AAB - للنشر على Google Play)
```bash
flutter build appbundle --release
```

### للآيفون (IPA)
```bash
# يتطلب جهاز Mac
flutter build ios --release
```

## 🔧 التكوين

### تعديل رابط API
في ملف `lib/core/services/api_service.dart`، قم بتعديل:
```dart
static const String baseUrl = 'https://your-api-url.com';
```

### إعداد الإشعارات
التطبيق يستخدم `flutter_local_notifications` للإشعارات المحلية.

### إعداد التخزين المحلي
التطبيق يستخدم `hive` و `shared_preferences` للتخزين المحلي.

## 📁 هيكل المشروع

```
lib/
├── core/
│   ├── routes/          # نظام التوجيه
│   ├── services/        # الخدمات (API, الصوت, الإشعارات, التخزين)
│   └── theme/           # الثيمات والألوان
├── models/              # النماذج (Article, Dua, etc.)
├── providers/           # مزودي الحالة (Provider)
└── screens/             # الشاشات والعناصر
    ├── home/            # الشاشة الرئيسية
    └── widgets/         # عناصر الواجهة
```

## 📝 ملاحظات مهمة

### للنشر على Google Play:
1. أنشئ حساب مطور ($25 لمرة واحدة)
2. قم بإنشاء signing key
3. ارفع ملف AAB
4. أضف لقطات الشاشة والوصف

### للنشر على App Store:
1. أنشئ حساب مطور Apple ($99 سنوياً)
2. يتطلب جهاز Mac للبناء
3. أنشئ شهادة توقيع
4. ارفع ملف IPA عبر App Store Connect
